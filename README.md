# Project-J
본 프로젝트는 네오위즈 퀘스트에서 주최하는 게임 제작 공모전 출품을 위한 게임 제작 프로젝트입니다.
게임 제작으로 Unity, 리소스 제작으로 Blender, 음악 제작으로 Flstudio Mobile를 사용하였습니다.

## Feature
- Unity로 엉뚱한 세계관과 스토리와 캐릭터로 흥미로운 게임을 제공합니다.
- Blender로 귀엽고 아기자기하고 앙증맞은 그래픽을 제공합니다.
- Flstudio Moblie로 적절한 음악을 제공합니다.

## Project Structure
```
Project-J/
├── docs/             # 이 디렉토리는 문서 관련 디렉토리입니다.
│ └── 게임 소개 자료.pdf

└── LICENSE
```

## Installation
1. 리포지토리를 클론합니다.
	```
	gh repo clone YooJunHyuk123/Project-J
	```

2. 프로젝트 디렉토리로 이동합니다.

3. 게임을 실행합니다.

## Usage
- 새 게임을 누르면 게임을 시작합니다.
- 십자키를 누르면 캐릭터가 상하좌우로 움직입니다.
- Q를 누르면 캐릭터가 근접 공격을 하고, E를 누르면 캐릭터가 원거리 공격을 합니다.

## Test
- 게임 실행, 그래픽 출력, 음악 재생이 정상적으로 동작하는지 테스트하였습니다. 
- 버그가 발생하는지 테스트하였습니다.
- 조작감이 불편한지, 게임 실행이 지나치게 느리지는 않은지 테스트하였습니다.
